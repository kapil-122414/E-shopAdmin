# Overview

This project is an **E-commerce Admin Panel** built with **React.js**. It provides a clean and user-friendly interface for managing the day-to-day operations of an online store.

The application allows administrators to manage products, categories, customers, and orders from a single dashboard. It is designed to work seamlessly with a REST API backend, making it easy to perform CRUD operations, monitor store activity, and keep product information up to date.

The project focuses on creating a responsive and organized admin experience with reusable React components, efficient state management, and a modern UI. It also demonstrates best practices for frontend development, including API integration, routing, form handling, and component-based architecture.

This admin panel was developed as a practical project to strengthen my frontend development skills and to understand how real-world e-commerce management systems are built.
